# tsconfig

Best practice of TypeScript configuration for React front-end projects

## Install

```
npm install --save-dev @rive/tsconfig/tsconfig.json
```

## Add to your tsconfig.json

```json
{
  "extends": "@tsconfig/recommended/tsconfig.json"
}
```
